## Document Categories in GK-Nexus

When uploading documents, use these categories:

| Category | When to Use |
|----------|-------------|
| `AGREEMENT` | Contracts, MOUs, service agreements |
| `AFFIDAVIT` | Sworn statements, statutory declarations |
| `CERTIFICATE` | Registration certs, completion certs |
| `APPLICATION` | Application forms, submissions |
| `FINANCIAL` | Bank statements, financial reports |
| `IDENTIFICATION` | IDs, passports, TIN documents |
| `TAX` | Tax returns, receipts, assessments |
| `IMMIGRATION` | Visas, permits, travel documents |
| `LEGAL` | Legal documents, court papers |
| `CORRESPONDENCE` | Letters, emails, communications |
| `SUPPORTING` | Supporting evidence, backup docs |
| `OTHER` | Anything else |

## File Requirements

### Accepted File Types
- PDF (preferred for final documents)
- DOC/DOCX (Microsoft Word)
- XLS/XLSX (Microsoft Excel)
- JPG/PNG (Images, scans)

### Size Limits
- Maximum file size: 10 MB per file
- No limit on number of files per matter

### Naming Conventions

**Recommended format:**
```
[ClientName] - [DocumentType] - [Date or Version].ext
```

**Examples:**
- `John Smith - Income Tax Return - 2024.pdf`
- `ABC Ltd - Certificate of Incorporation - Final.pdf`
- `Jane Doe - Passport Scan.jpg`

## Tips for Staff

### Document Preparation
1. Always use current form versions
2. Double-check calculations
3. Verify client information matches ID
4. Get proper signatures where required
5. Keep copies of all submitted documents

### Client Document Requests
1. Be specific about what you need
2. Explain why each document is required
3. Set reasonable deadlines
4. Follow up on missing items
5. Acknowledge receipt promptly
