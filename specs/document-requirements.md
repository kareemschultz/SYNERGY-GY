## Universal Client Documents

These documents are required for client profile creation and may be referenced across multiple services:

| Document Type | Description | Required | Notes |
|--------------|-------------|----------|-------|
| National ID | Guyana National Identification Card | Yes | Must be valid and not expired |
| Passport | Valid passport | Conditional | Required for immigration services |
| Birth Certificate | Original or certified copy | Yes | Must be government-issued |
| Proof of Address | Utility bill, lease agreement, or bank statement | Yes | Must be dated within 3 months |
| TIN Certificate | Tax Identification Number certificate | Yes | Required for all tax-related services |
| NIS Number | National Insurance Scheme number | Yes | Required for employment-related services |
| Photographs | Passport-size photos | Conditional | Required for specific applications |

## Document Upload Specifications

### File Format Requirements

**Accepted Formats:**
- **Documents:** PDF (preferred), DOCX, DOC
- **Images:** JPG, JPEG, PNG
- **Scans:** PDF, TIFF

**Not Accepted:**
- Executable files (.exe, .bat, .sh)
- Compressed archives (without prior approval)
- Encrypted files (without decryption instructions)

### File Size Limits

| Document Type | Maximum Size | Notes |
|---------------|--------------|-------|
| Standard documents | 10 MB | Per file |
| High-resolution scans | 25 MB | Special approval required |
| Bulk uploads | 100 MB | Total per upload session |

**Optimization Guidelines:**
- Scan documents at 300 DPI for text documents
- Use 600 DPI for documents requiring high detail
- Convert color scans to grayscale when color not necessary
- Use PDF compression tools if file exceeds limits

### Naming Conventions

**Standard Format:**
```
[ClientID]-[ServiceType]-[DocType]-[Date]
```

**Examples:**
- `CL00123-GCMC-Passport-20251211.pdf`
- `CL00456-KAJ-TaxReturn-2024.pdf`
- `CL00789-GCMC-EmploymentContract-20251211.pdf`

**Field Definitions:**
- **ClientID:** Unique client identifier (e.g., CL00123)
- **ServiceType:** GCMC or KAJ
- **DocType:** Brief description (e.g., Passport, TaxReturn, NationalID)
- **Date:** Upload date or document date (YYYYMMDD)

**Additional Guidelines:**
- Use hyphens (-) as separators, not spaces or underscores
- Use PascalCase for DocType (e.g., EmploymentContract)
- Include version numbers if submitting revisions (e.g., -v2)
- Maximum filename length: 100 characters

### Required Metadata

Each uploaded document must include:

| Field | Description | Required | Example |
|-------|-------------|----------|---------|
| Client ID | Unique client identifier | Yes | CL00123 |
| Document Type | Category of document | Yes | Passport |
| Service Type | GCMC or KAJ service | Yes | GCMC-Immigration |
| Upload Date | Date of upload | Auto | 2025-12-11 |
| Document Date | Date on document | Yes | 2024-06-15 |
| Expiry Date | If applicable | Conditional | 2030-06-15 |
| File Size | Size in MB | Auto | 2.5 MB |
| Uploaded By | Staff member | Auto | User123 |
| Verification Status | Document verification | Yes | Pending/Verified/Rejected |
| Notes | Additional information | Optional | Renewed passport |

### Quality Requirements

**Scanned Documents:**
- All pages must be readable
- No missing pages or portions
- Correct orientation
- No excessive shadows or glare
- Text must be legible at 100% zoom

**Photographs:**
- Clear focus
- Adequate lighting
- No filters or alterations
- Complete document visible
- Minimum resolution: 1200x1600 pixels

**Digital Documents:**
- Original format when possible
- Not screenshots of documents
- Editable PDFs preferred for forms
- Include all pages

### Security Requirements

**Before Upload:**
- Verify document is virus-free
- Ensure no malicious code embedded
- Check file integrity

**During Upload:**
- Use encrypted connection (HTTPS)
- Verify upload completion
- Confirm file checksum

**After Upload:**
- Document stored in encrypted storage
- Access logged and audited
- Backup created automatically

### Validation Checks

**Automated Validation:**
- File format verification
- File size compliance
- Virus scanning
- Metadata completeness
- Naming convention compliance
- OCR for searchability (PDF)

**Manual Validation:**
- Document authenticity
- Information accuracy
- Expiry date verification
- Signature verification
- Completeness check

## Document Retention Policy

### Retention Periods by Document Type

#### Tax-Related Documents
**Retention Period:** 7 years from filing date

**Includes:**
- Income tax returns (individual and corporate)
- VAT returns
- Withholding tax records
- Tax receipts and payment confirmations
- Expense receipts supporting tax deductions
- Financial statements
- Bank statements
- Payroll records
- NIS contribution records

**Reason:** Required by Guyana Revenue Authority regulations

#### Legal Documents
**Retention Period:** 10 years minimum (some permanent)

**Includes:**
- Property deeds and titles (Permanent)
- Wills and estate documents (Permanent)
- Company incorporation documents (Permanent)
- Contracts and agreements (10 years after expiration)
- Powers of attorney (10 years or until revoked)
- Court documents (Permanent)
- Affidavits (10 years)

**Reason:** Legal requirement and potential future reference

#### Immigration Documents
**Retention Period:** 5 years after expiry/completion

**Includes:**
- Work permits
- Visa applications
- Citizenship applications
- Residency permits
- Supporting documentation

**Reason:** Immigration department requirements and appeal periods

#### Business Registration Documents
**Retention Period:** Permanent for active entities, 10 years after dissolution

**Includes:**
- Certificates of incorporation
- Business name registrations
- Partnership agreements
- Annual returns
- Amendment certificates
- Dissolution documents

**Reason:** Legal requirement and proof of business history

#### Employment Records
**Retention Period:** 7 years after employment ends

**Includes:**
- Employment contracts
- Personnel files
- Performance reviews
- Disciplinary records
- Termination documentation
- Training certificates

**Reason:** Labor law requirements and potential disputes

#### Client Identity Documents
**Retention Period:** 5 years after service completion

**Includes:**
- National IDs (copies)
- Passports (copies)
- Birth certificates (copies)
- Proof of address
- Photographs

**Reason:** KYC compliance and verification

#### Financial Records (Non-Tax)
**Retention Period:** 7 years

**Includes:**
- Invoices (issued and received)
- Receipts
- Bank reconciliations
- Asset registers
- Loan documentation

**Reason:** Accounting standards and audit requirements

#### General Correspondence
**Retention Period:** 5 years

**Includes:**
- Email correspondence
- Letters
- Memoranda
- Meeting minutes
- Service requests

**Reason:** Reference and continuity of service

### Storage Requirements

#### Physical Documents
- Climate-controlled storage
- Fire-proof cabinets for critical documents
- Organized by client and document type
- Indexed for easy retrieval
- Access restricted to authorized personnel

#### Digital Documents
- Encrypted cloud storage
- Regular backups (daily incremental, weekly full)
- Off-site backup storage
- Version control for updated documents
- Audit trail of access and modifications

### Disposal Procedures

#### When Retention Period Expires

**Physical Documents:**
1. Verify retention period has elapsed
2. Obtain client consent if required
3. Create disposal log
4. Shred using cross-cut shredder
5. Certificate of destruction for sensitive documents

**Digital Documents:**
1. Verify retention period has elapsed
2. Create deletion log
3. Secure deletion (unrecoverable)
4. Verify deletion completion
5. Update document management system

#### Exceptions to Disposal
Documents should NOT be disposed if:
- Active legal proceedings involving the document
- Client specifically requests retention
- Under investigation or audit
- Part of permanent record
- Historical significance

- Quarterly review of retention compliance

- Verification of backup integrity
- Access log review
- Storage condition checks

## Document Validation Rules

### Identity Document Validation

#### National ID
- Must be valid (not expired)
- Photo clearly visible and matches client
- ID number format: valid Guyana National ID format
- All fields legible
- Not damaged or altered
- Both sides scanned/copied

#### Passport
- Must be valid with minimum 6 months remaining validity (for immigration services)
- Biographical page clearly visible
- Passport number matches other documents
- Photo matches client
- All visa pages included if relevant
- MRZ (Machine Readable Zone) legible

#### Birth Certificate
- Government-issued or certified copy
- Legible stamp/seal visible
- Issued by recognized authority
- Translation required if not in English
- Apostille required for international use

### Financial Document Validation

#### Bank Statements
- Original or certified bank stamp
- Consecutive months with no gaps
- Account holder name matches client
- Address visible (if required)
- All pages included
- Recent (as specified by service)

#### Financial Statements
- Properly formatted (balance sheet, P&L, cash flow)
- Signed by responsible person
- Auditor's report if required
- Consistent with tax filings
- Complete notes to accounts

### Property Document Validation

#### Title Deeds
- Issued by land registry
- Legal description matches property
- Owner name matches seller
- All encumbrances noted
- Seal/stamp visible
- All pages included

#### Survey Plans
- Licensed surveyor signature
- Date within acceptable period
- Boundaries clearly marked
- Scale indicated
- Matches legal description
- Government approval stamp

### Business Document Validation

#### Company Registration
- Issued by Registrar of Companies
- Company name matches other documents
- Registration number visible
- Not struck off or dissolved
- Certified copy if not original
- Current (not historical) status

#### TIN Certificate
- Issued by Guyana Revenue Authority
- TIN format valid
- Name matches business/individual
- Not revoked or suspended
- Clear and legible

### Professional Document Validation

#### Educational Certificates
- Issued by recognized institution
- Authentication/apostille if foreign
- Not altered or damaged
- Clearly shows qualification obtained
- Date of award visible
- Translation if not in English

#### Professional Licenses
- Issued by recognized professional body
- Current (not expired)
- Name matches applicant
- Discipline/restrictions noted
- Verification stamp visible

### Document Authentication

#### Certified Copies
- "Certified True Copy" stamp
- Certifier's signature
- Certifier's name and designation
- Date of certification
- Certifier's seal (if applicable)
- Contact information of certifier

**Acceptable Certifiers:**
- Justice of the Peace
- Attorney-at-Law
- Notary Public
- Police Officer (Inspector or above)
- Magistrate
- Commissioner of Oaths

#### Apostille Requirements
For documents to be used internationally or originating from abroad:
- Apostille certificate attached
- Issued by competent authority of origin country
- Recent (within acceptable period)
- Document and apostille match
- Translated if necessary

### Common Rejection Reasons

Documents may be rejected if:
1. Expired or outside valid period
2. Poor quality (illegible, cut off, blurry)
3. Missing pages
4. Altered or appears tampered with
5. Name mismatches
6. Not properly certified
7. Missing required stamps/seals
8. Wrong document type
9. Insufficient evidence
10. Not translated when required
11. File corrupted or cannot open
12. Wrong file format
13. Exceeds size limits
14. Contains viruses or malware
15. Incomplete information

### Document Verification Process

1. **Initial Review**
   - File quality check
   - Format compliance
   - Naming convention
   - Metadata completeness

2. **Content Verification**
   - Document type confirmation
   - Expiry date check
   - Information accuracy
   - Completeness check

3. **Authenticity Verification**
   - Certification validation
   - Stamp/seal verification
   - Signature verification
   - Cross-reference with issuing authority (if needed)

4. **Compliance Check**
   - Meets service requirements
   - Matches client information
   - Satisfies regulatory requirements
   - Appropriate for purpose

5. **Final Approval**
   - Verification stamp in system
   - Document linked to client file
   - Document linked to service request
   - Notification sent to client

## Notes and Best Practices

### For Clients

1. **Submit Early:** Provide documents as early as possible to allow time for verification and correction if needed.

2. **Original Documents:** Bring original documents for verification when submitting certified copies.

3. **Validity Period:** Ensure documents have sufficient validity period remaining.

4. **Completeness:** Submit all required documents together to avoid delays.

5. **Clarity:** Ensure scans/photos are clear, complete, and legible.

6. **Updates:** Notify immediately if any document information changes.

### For Staff

1. **Checklist Use:** Use service-specific checklists for document requirements.

2. **Verification:** Verify all documents promptly upon receipt.

3. **Communication:** Communicate clearly with clients about any issues or missing documents.

4. **Security:** Handle all client documents with strict confidentiality.

5. **Organization:** Maintain organized document files for easy retrieval.

6. **Updates:** Keep informed of any changes to document requirements or regulations.

### Common Issues and Solutions

| Issue | Solution |
|-------|----------|
| Expired documents | Request renewal before proceeding |
| Poor scan quality | Request rescan with better resolution |
| Missing pages | Request complete document |
| Name mismatch | Request explanation or name change documentation |
| Foreign language | Request certified translation |
| Unsigned documents | Return for signature |
| Incomplete forms | Return with instructions for completion |
| Wrong document type | Clarify requirements and request correct document |
