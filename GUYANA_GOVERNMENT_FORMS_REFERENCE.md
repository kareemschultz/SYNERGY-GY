## üîß Auto-Fill Field Mapping

### Client Fields Available
```typescript
// Individual Client
firstName, lastName, middleName
dateOfBirth, nationality, gender
address, city, country
email, phone
tinNumber, nisNumber, nationalId, passportNumber

// Corporate Client  
companyName, tradingName
registrationNumber, tinNumber, nisNumber
registeredAddress, businessAddress
incorporationDate, businessType
directors[], shareholders[]
```

### Form Field Mapping Example
```typescript
const R400F1_MAPPING = {
  "Surname": "lastName",
  "First Name": "firstName", 
  "Date of Birth": "dateOfBirth",
  "Address": "address",
  "TIN (if any)": "tinNumber",
  "National ID": "nationalId"
};
```
